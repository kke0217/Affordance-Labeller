# 프로젝트 개요

## 프로젝트명
어포던스 라벨링 툴 및 데이터 관리 시스템 구축

## 연구 맥락
본 프로젝트는 휴머노이드 로봇의 양손 파지 조작을 위한 모방학습 및 강화학습 활용을 목표로 한다. 이번 단계의 현실적 목표는 완전 자동화가 아니라, 연구원이 실제로 라벨을 만들고 관리할 수 있는 최소 동작 시스템(MVP)을 확보하는 것이다.

## 이번 달 핵심 목표
- YCB mug 1개 이상을 RGB-D 또는 mesh 형태로 로드
- body / handle / rim / interior 파트 구분
- affordance 영역 수동 라벨링
- contact region mask 지정
- candidate 6D grasp/action pose 저장
- semantic action tag 저장
- JSON 저장 / 재로드 / 검수 상태 관리

## 기술 원칙
- 프론트엔드: Viser 기반 웹 UI
- 처리 백엔드: Open3D
- 저장 구조: 내부 JSON v0.1
- 설계 방향: RLDS / LeRobot 변환 가능 구조 확보

## 이번 달 제외 범위
- 완전 자동 affordance 추출
- Isaac Sim 연동 및 물리 안정성 자동 검증
- RLDS / LeRobot 완전 직출력
- 다중 사용자 협업
- DB 백엔드 구축
- 실로봇 연동
- PartNet-Mobility 통합
